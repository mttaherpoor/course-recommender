# 📚 Course Recommender (git.ir)

A simple course recommendation system that helps users find the most relevant and best-matching courses available on **git.ir** based on their preferences and data-driven logic.

---

## 🚀 Overview

This project is designed to help users:

* Discover better course options
* Avoid irrelevant or low-quality choices
* Save time in searching manually
* Get structured recommendations from available course data

It can be extended into a more advanced recommendation engine using NLP or user behavior analysis.

---

## ⚙️ Features

* 🔍 Course data processing and filtering
* 📊 Basic recommendation logic
* 🧠 Extensible structure for ML/NLP improvements
* 📁 Clean and modular Python codebase
* 📦 Easy to run and modify

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/mttaherpoor/course-recommender.git
cd course-recommender
```

Create virtual environment (optional but recommended):

```bash
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

fill excel files like example.

Run the main script:
